<h1> Live to GAME - PowerBI</h1>

- Projeto desenvolvido na disciplina de Estatística - Profª Maria Luisa Cervi Uzun | Fatec Franca - Faculdade de Tecnologia de Franca Dr Thomaz Novelino;
